FAQ

O que é HTML?
+

Linguagem de Marcação de Hypertexto (HTML - Hypertext Markup Language) é o código que você usa para estruturar seu conteúdo web, dando significado e propósito.

O que é CSS?
+

Folhas de Estilo em Cascata (CSS - Cascading Stylesheets) é o código que você usa para dar estilo ao seu site.

O que é JavaScript?
+

JavaScript é a linguagem de programação que você usa para adicionar recursos interativos ao seu site.
